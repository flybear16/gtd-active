# ✅ gtd-active - GTD 任务管理 skill

> 基于 David Allen《Getting Things Done》方法论的 AI 自动化任务管理系统

## 🎯 核心理念

> 你的大脑是用来思考的，不是用来存储的。

GTD 将一切事务从大脑中清空，放入可信赖的系统，让你专注于执行。

## 📋 五步流程

1. **📥 收集（Capture）** — 把所有想法、任务、信息无差别塞进收件箱
2. **🔍 处理（Clarify）** — 问"这是什么？""需要行动吗？"
3. **📂 组织（Organize）** — 归类到项目 / 下一步行动 / 日程 / 等待 / 将来/也许 / 参考资料
4. **🔁 回顾（Reflect）** — 每周检视所有清单
5. **🚀 行动（Do）** — 基于情境（时间/地点/精力/优先级）选择下一步

## ✨ 技能能力

- 自动归类（基于任务描述语义）
- 手动归类（用户指定清单）
- 列出某清单所有任务
- 完成任务 + 标记完成时间
- 每日 / 每周回顾模板

## 🔧 技术栈

- **格式**：纯 SKILL.md（OpenClaw / Hermes 兼容）
- **存储**：本地 JSON / Markdown
- **触发**：自然语言「把这个归到下一步行动」
